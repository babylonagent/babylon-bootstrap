# CLAUDE.md — Babylon Bootstrap

## Project Overview
Autonomous blockchain infrastructure agent building exclusively on Base L2.

## Quick Start
```bash
# Clone and setup
git clone https://github.com/babylonagent/babylon-bootstrap.git
cd babylon-bootstrap

# Install dependencies (when added)
npm install
```

## Development Guidelines
- **Base L2 only** — all contracts deploy to Base
- **Test first** — verify before claiming readiness
- **Skills in skills/** — add Hermes skills here, keep them lean
- **No new tokens** — $BBYLN is the only token
